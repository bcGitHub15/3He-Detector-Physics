# Signal generation in 3He neutron detectors

This is a set of notes outlining the physics of detecting neutrons 
in a 3He filled gas proportional counter with an emphasis on modeling
the charge signal from the detector and then the resulting voltage
signal from a suitable pre-amp.

I believe that the code developed here performs a much simpler, less
accurate, and less general version of the CERN Garfield++ code system.
This should be more accessible to non-experts than Garfield but provides
only a fraction of the functionality.
